# Development Ground Rules

## Language Preferences

- **Python** — Default for prototyping and general development
- **TypeScript/Node** — Use only for web apps when there's a good reason

## Python Environment

- **Always use `uv`** for virtual environment management
- No exceptions — `uv` is the standard tool

## Project Structure

- **Root directory:** `~/playground`
- **Per-project subdirectories:** `~/playground/<project-name>/`

## Examples

```
~/playground/
├── blender-mcp/
├── vintage-mac-tools/
├── homestuck-archive/
└── ev-nova-mod/
```
